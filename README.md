# Bearbrick Backend

Bare-minimal Express backend starter.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env`:
```bash
cp .env.example .env
```

3. Start the server:
```bash
npm start
```

For development with auto-reload:
```bash
npm run dev
```

## Endpoints

- `GET /` - Basic health check
- `GET /health` - Health status
